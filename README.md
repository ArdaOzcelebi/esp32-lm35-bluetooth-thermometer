# esp32-lm35-bluetooth-thermometer
ESP32 tabanlı LM35DZ ile sıcaklık termometresi. RC filtre + oversampling + EMA filtre kullanılarak stabil sıcaklık ölçümü yapılır ve Bluetooth üzerinden mobil cihaza aktarılır.

Donanım:

- ESP32 NodeMCU-1.1
- LM35DZ temperature sensor
- 1kΩ resistor
- 100nF capacitor
- Breadboard + jumper wires
